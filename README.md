# Проект 7 спринта - написание автотестов для API учебного сервиса «Яндекс.Самокат».
Структура проекта:
- allure_results - папка с результатами тестирования
- tests - папка с тестами
- data.py - файл с данными
- endpoints.py - файл с ручками
- helper.py - файл с методом для генерации нового курьера 
- requirements.txt - файл с библиотеками

Команды для запуска тестов:

Установка зависимостей

pip install -r requirements.txt

Запуск автотестов без отчета

pytest -v

Запуск автотестов и создание отчета о тестировании в Allure

pytest --alluredir=allure_results

Генерация отчета из результатов тестов 

allure serve allure_results
